# OOP 과제



### 다음과 같은 속성과 행위를 가지는 클래스 만들기

* 강아지
  * 속성: 이름, 나이, 몸무게, 견종
  * 행위: 짖기, 먹기
```swift
class Dog {
    let name: String = ""
    var age: Int = 0
    var weight: Double = 0
    let kind: String = ""
    
    func bark(){
        print("멍")
    }
    func eat(){
        print("냠")
    }
}
```

* 학생
  * 속성: 이름, 나이, 학교명, 학년
  * 행위: 공부하기, 먹기, 잠자기
```swift
class Student {
    let name = ""
    var age = 0
    var schoolName = ""
    var grade = 0
    
    func study(){
        print("공부")
    }
    func eat(){
        print("eat")
    }
    func sleep(){
        print("sleep")
    }
}
```
